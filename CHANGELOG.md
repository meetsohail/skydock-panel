# Changelog

All notable changes to SkyDock Panel will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-01-XX

### Added
- Initial release of SkyDock Panel
- Server monitoring (CPU, RAM, disk, uptime)
- Service management (Nginx, Apache, MySQL)
- Website management (PHP and WordPress)
- SSH profile configuration
- One-click WordPress installation
- Modern Tailwind CSS UI
- Systemd service integration
- Automated installation script
- Django REST API endpoints

### Security
- Encrypted SSH credential storage
- Session-based authentication
- Secure command execution utilities
